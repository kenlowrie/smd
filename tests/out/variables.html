Purpose: Testing the @var namespace

Variables provide a convenient means for text substitution. The syntax for defining a variable is: @var variable="value". Take the following example:

@var name="Ken Lowrie"

Now, anywhere I write [name], it will be replaced with "Ken Lowrie". Let's do that now: Ken Lowrie <-- Should be Ken Lowrie.

If I instead write: [name]=[*Ken Lowrie*], then everywhere I write [name], it will be replaced with <em>Ken Lowrie</em>. Okay, let's go ahead and do that now.

And now, Ken Lowrie <-- should be Ken Lowrie wrapped with <em> tags.
